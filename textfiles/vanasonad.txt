Head kakud süüakse ära, santidega visatakse ratast ;
Kui om hää mees, saad süüa, kui halb, ei saa sedagi	;
Mis enne jõulu tihase, seda jõulupäe süüase;
Ei kahe pere koer saa elades süüa	;
Kui kastet viia, siis soab ikke pudru süüa	;
Kiideldes koeraliha süüakse	;
Kiida koerale liha süüa	;
Kui kirves veel alles, saab ikki süüa: kirves patta ja vars paja alla	;
Kus kits kinni, säält ta peab süüa ka saama	;
Peaks kivi nõnda süüa aitama, kut vesi juua aitab, siis poleks tööd tarvis teha olavad;	
Anna siis koerale süüa, kui koer karja lähab	;
Koer haugub sinna poole, kust süüa saab	;
Kudas koeral süüa antakse, nõnda ta käib karjas	;
Kuida koer süüa saab, nenda koer haugub	;
Seda kohta ei leia, kus magades süüa saab ja riiet kanda	;
Laisk ei tohe leiba süüa, / leib ei ole muidu leida	;
Antud leib süüakse ja andja käsi põlatakse ära	;
Parem leivakooruke kodu närida kui saia süüa võõra juures	;
Iga lind saab oma laulu eest süüa	;
Lõunat võid süüa rikka man, õhtu maga vaese man	;
Mihklipäeval süüakse lusikaga võid	;
Kui muna mõstad süüa, küll sis koore ka mõstad katki tiha	;
Narrile anna leib kätte, ta ei oska süüa	;
Kes noorelt nobe koguma, sel vanuni vara süüa	;
Noore hobusele ja lapsele anna süüa ja peksta	;
Nõuandja on suurem süüalune kui kurjategija	;
Kuida näpud jõuavad liikuda, sedamööda suu jõuab süüa	;
Oma keeta, oma süüa	;
Oma tehtud kaapsad oo hapud süüa;
